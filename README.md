# iOS-NoSoundCamera

撮影音なしで撮影ができる
カメラ撮影用サンプルアプリ


# Requirement

Xcode: Version 14.2 (14C18)
iOS: 16.0~

# Installation


```bash
git clone https://github.com/9610r/iOS-NoSoundCamera.git
```

# Note

写真はライブラリに保存される。
